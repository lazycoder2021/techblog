How to Create a JS Image Slider?

Let's create a simpler version of image slider you come across websites.

HTML & CSS Part

We'll be writing html, css and js in the same file. So a single .html will do.

HTML Skeleton

We need 3 divs: one for the image, one for buttons, and a wrapper for both.

Centering the Container

We use CSS flex property to center the elements on screen.

Styling the Image Holder Div

Let's give a border to the image container div so that it appears on screen.

Note: margin:auto is not needed. align-items: center is enough to center the divs.

Fitting Image Into Image Holder Div

We then ensure the image does not overflow its container by giving it 100% of the size of its container; in other words, the same size as that of its container.

Accessing HTML Elements

Time to add javascript between script tags and gain access to button elements.

We use document.querySelector() to select the buttons.

Adding Event Listeners

Next add 'click' event listener to the buttons to help us switch or change the images.

Hardcoding The Image

Select the image element using its class 'image' and set its src property to '1.jpg'

Note: You should have image file named '1.jpg' in your working folder

Using Array to Store/Access Images

Let's give the images 'numbers' as their names to simplify things.

The extension of our images (.jpg) will be concatenated later using js to render them.

Changing Images Dynamically Using Counter

Declare a counter variable with '0' value

Increment the counter whenever the next button is clicked

Decrement the counter whenever the prev button is clicked

Then pass counter as the index value to 'images' array for selecting the image.

Use if condition to ensure the counter does not exceed the length of our array.

0 and images.length - 1 are the starting and ending points of our array respectively.

Automating Our Slider

This can be done using setInterval function.

Ensuring Buttons Override Automation

We just have to clear the setInterval function within 'click' events of our buttons.
